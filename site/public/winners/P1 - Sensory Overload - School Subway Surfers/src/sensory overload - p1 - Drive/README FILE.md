# **Endless Hallway Runner**

A  endless runner built with **Python** and **Pygame**. Dodge obstacles, collect assignment pages, and survive for as long as possible as the game gradually increases in speed and difficulty.

---

## **Features**

* Endless runner gameplay with progressive difficulty  
* Multiple environments that change as the player advances  
* Collectible assignment pages used for scoring and upgrades  
* Shop system with unlockable skins and power-ups  
* Procedurally generated obstacles  
* Real-time synthesized audio without external sound files

---

## **Gameplay**

* The player automatically runs forward while avoiding obstacles and collecting assignment pages.  
* As the score increases, the game becomes faster and introduces new visuals and obstacle patterns.  
* Assignment pages increase the player's score and can also be spent in the shop.  
* The game ends when the player collides with an obstacle.

---

## **Controls**

* **W / ↑** \- Jump  
* **S / ↓** \- Slide  
* **R** \- Pause / Restart  
* **ESC** \- Quit Game

---

## **Built With**

* Python  
* Pygame  
* Object-Oriented Programming (OOP)

---

## **Project Goal**

* Recreate the fast-paced gameplay of classic endless runners in a 2D environment.  
* Explore concepts such as game physics, procedural generation, audio synthesis, and player progression systems.
